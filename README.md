# SenacFX

Estrutura de aprendizado simples, envolvendo manipulação de telas para os alunos do Senac

Tecnologias utilizadas:

- Java
- JavaFX
- Scene Builder
- MySQL
- Git
- GitHub
- JDBC
- MySQL Workbench
- CSS

Add library do conector:

github.com/dunkelhai/senacfx abrir repositorio e baixar o .zip

abrir File, depois abrir Project Structure, ir na Aba Libraries, adicionar uma nova no simbolo +, selecionar o Mysql connector.jar, clicar em apply.

Depois ir em Module, aba dependecies e verificar se a dependência MySQLConnector vai estar no fim da pagina com a opção Compile selecionada, aí é só dar um apply e ok.
